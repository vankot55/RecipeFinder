Роль: Ти — досвідчений UI/UX дизайнер, який працює над вдосконаленням платформи для продавців маркетплейсу Rozetka.

Завдання: Оновити існуючі макети "Seller Hub", перетворивши їх зі статичної навчальної платформи на проактивного, персоналізованого помічника для продавців. Потрібно зберегти фірмовий стиль Rozetka (кольори, шрифти, загальна чистота дизайну), але додати нові функціональні блоки та елементи.

Вхідні дані:

Макет 1 (Головна сторінка "Seller Hub"): Містить привітання, пошук, основний навчальний курс "Мій шлях навчання" з прогрес-баром, та рекомендовані курси.

Макет 2 (Сторінка навчання): Містить історію навчання, ключові теми та найближчі вебінари.

Ключові зміни для впровадження:

1. Поглиблення персоналізації та інтеграція з даними (Макет 1):

Додай новий блок під назвою "Можливості для вашого магазину".

Цей блок має містити персоналізовані, керовані даними картки-рекомендації. Створи візуальні приклади для таких карток:

Картка-попередження: "Ми помітили, що ваші відправки затримуються. Перегляньте курс по оптимізації логістики, щоб покращити свій рейтинг."

Картка-порада: "Конверсія товару [Назва товару] нижча за середню. Дізнайтеся, як створити продаючі фотографії."

Картка-можливість: "Ви ще не використовуєте Промокоди. Пройдіть короткий туторіал та залучіть більше покупців."

2. Впровадження гейміфікації (Макет 1 та 2):

На головній сторінці (Макет 1), поруч з іменем продавця або в профілі, додай іконку для "Мої досягнення".

Створи окремий візуальний блок або сторінку "Досягнення", де будуть відображатись зароблені бейджі (наприклад, "Майстер SEO", "Експерт з фотоконтенту", "Гуру роботи з клієнтами"). Намалюй 3-4 приклади таких бейджів.

В "Історії навчання" (Макет 2) біля повністю завершених курсів додай кнопку "Завантажити сертифікат".

3. Розширення форматів контенту (Макет 1):

Додай на головну сторінку невеликий горизонтальний блок "Швидкі поради" або "Seller Shorts". Це має бути карусель з прев'ю коротких (1-2 хв) відео-інструкцій.

Покажи, як пошук тепер обробляє різні типи контенту. При введенні запиту "повернення товару" у випадаючому списку результатів мають з'явитись іконки, що позначають тип контенту: [Курс], [Стаття з бази знань], [Відео], [Шаблон].

4. Покращення UX/UI та навігації (Макет 2):

На картках "Найближчі вебінари" додай статус-теги: LIVE для поточних, Запис для минулих.

Для вебінарів, що вже відбулися, зміни кнопку з "Зареєструватися" на "Переглянути запис".

Зроби блок "Ключові теми" більш інтерактивним. Кожна тема (напр., "Робота з товарами") — це клікабельна картка, яка веде на окрему сторінку. Створи макет такої сторінки, де зібрані всі матеріали за цією темою (курси, статті, вебінари) з можливістю фільтрації за форматом та складністю.

Очікуваний результат:
Надай оновлені версії двох макетів, що включають всі перераховані вище зміни. Макети мають бути візуально зрозумілими, логічно структурованими та демонструвати новий, більш динамічний та корисний функціонал Seller Hub.
